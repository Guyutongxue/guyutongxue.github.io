## 游戏

### 戏精的自我修养

主持人：王妍霏

介绍：

1. 自由分组，每组6个人
2. 每组都会供应6杯饮料，其中一杯是加了芥末/酱油/醋等等的黑暗料理
3. 由其他组来猜，喝到黑暗料理的是谁
4. 没喝到黑暗料理的可以假装自己喝到了,喝到黑暗料理的也可以假装自己没喝到
5. 总之就是锻炼戏精，我觉得这个游戏在咱们班应该很好进行。
